# FunPay Sales Analyzer

Расширение для Chrome/Edge, которое отслеживает оплаченные, но не закрытые заказы FunPay.

## Возможности

- анализ за 1, 7, 30 дней или свой период от 0 до 30 дней;
- `0` дней означает анализ без ограничения по дате в пределах загруженных страниц;
- основной отчёт считает только заказы со статусом `оплачен`, которые ещё не закрыты, не завершены и не отменены;
- показывает количество неподтверждённых заказов и их общую стоимость;
- показывает среднюю стоимость неподтверждённого заказа;
- строит график неподтверждённых заказов по дням;
- сравнивает неподтверждённые заказы за 7 и 30 дней;
- ищет в чатах оплаченных заказов поддержку, арбитраж, модератора или администрацию;
- выделяет найденные проблемные заказы в списке продаж на странице FunPay;
- показывает товары, по которым чаще встречаются проблемные чаты;
- сохраняет локальную историю последних 20 анализов;
- проигрывает звук, если найден новый проблемный оплаченный заказ;
- экспортирует отчёт в CSV.

## Оптимизация

- страницы заказов кэшируются на время открытой вкладки, поэтому повторный анализ работает быстрее;
- чаты проверяются параллельно, но с лимитом, чтобы не перегружать страницу;
- период считается календарными днями: 7 дней = сегодня и 6 предыдущих дней;
- popup использует версионированный обмен с content script, чтобы после обновления не отвечал старый код.

## Как установить

1. Откройте `chrome://extensions/` или `edge://extensions/`.
2. Включите режим разработчика.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите эту папку: `C:\Users\offsi\OneDrive\Документы\proect\расширение для funpay`.

## Как пользоваться

1. Войдите в FunPay.
2. Откройте раздел `Продажи`.
3. Нажмите на иконку расширения.
4. Выберите период и нажмите `Анализировать`.

Расширение не отправляет данные на сторонние серверы. Оно читает открытую страницу FunPay и страницы заказов через ваш текущий вход в аккаунт.
